Design System Rules

Documenting Dark Mode Red Accents & Retro Typography

01. Typography System

Headings & Hero

VT323 Font

Applied to H1-H6, Metric Values, and .font-sans

Rule Implementation

font-weight: 400 !important;

Crucial: Prevents browser faux-bold blurring on pixel glyphs.

Body & Technical

Space Mono

Applied to paragraphs, lists, and metadata.

"The contrast between narrow pixel headers and wider monospace body text creates the core 'Terminal-Chic' identity."

02. Dark Mode Color Palette

Section Headings

Featured Work

Headings use accent-300 at 400 weight. It provides a warm, organic rose tone that avoids harsh neon vibrating against dark backgrounds.

Metric/KPI Values

4.5× ROAS

Key metrics use accent-400. This vivid red draws the eye immediately to performance data and success indicators.
